This repo contains an experimental early version of my custom DVCS - DITch.

"_DITch GIT, Store it with DIT_"

Based off often-painful experiences with the Git tools, I bagn researching why people so frequently find the tool complicated and hard to use.

Based off two key papers by Santiago Perez De Rosso et al. I designed a lightweight version control system, making the sacrifice between complicated features and a simpler usage pipeline.
